[2:] 슬라이싱을 하면 인덱스 2번째부터 끝까지 가져옴

    ex>x = ['a','b','c','d','e'] x[2:] >>> ['b','c','d','e']

[:2] 인덱스 처음부터 특정위치 즉, 2번째까지 가져옴

    ex>x = ['a','b','c','d','e'] x[:2] >>> ['a','b']

[2:4] 인덱스 특정위치부터 특정위치까지

    ex>x = ['a','b','c','d','e'] x[2:4] >>> ['b','c','d']
